3. HOW TO USE
---------------------------------------------------
1. Extract the template files to your project folder.
2. Open `index.html` in your browser to view the demo.
3. Replace images in `assets/images/` with your own images.
4. Customize text, colors, and layout by editing `assets/css/style.css` and `index.html`.
5. JS functionalities (smooth scroll, AOS, Swiper) are in `assets/js/script.js`.
6. Ensure Bootstrap, AOS, and Swiper libraries are included via CDN as in `index.html`.

---------------------------------------------------
4. IMPORTANT NOTES
---------------------------------------------------
- This template is purely HTML/CSS/JS. No backend functionality included.
- All icons use Font Awesome 6 via CDN.
- Make sure to keep the folder structure intact for CSS, JS, and image paths to work.
- Swiper & AOS libraries are already linked via CDN.
- Template is ready for Envato/ThemeForest submission.
- Images included are for demo purposes. Replace with your own for production.
